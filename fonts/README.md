# Manrope Font Files

This directory should contain the Manrope font files for the Roomix app.

## Required Font Files

Please download the Manrope font family from [Google Fonts](https://fonts.google.com/specimen/Manrope) and place the following files in this directory:

- `Manrope-Regular.ttf` (Weight: 400)
- `Manrope-Medium.ttf` (Weight: 500)
- `Manrope-SemiBold.ttf` (Weight: 600)
- `Manrope-Bold.ttf` (Weight: 700)
- `Manrope-ExtraBold.ttf` (Weight: 800)

## Alternative: System Font

If you don't have the Manrope font files, the app will fall back to the system default font (Roboto on Android, San Francisco on iOS).
